Connect Your MCP Servers

Second Brain connects to two MCP servers: GitHub (live repo activity) and XC Brain (persistent file storage). Add them in claude.ai Settings → Connectors.

GitHub not connected

Connects to your repos — live PRs, issues, commits in the Activity Feed.

1. Go to claude.ai → Settings → Connectors
2. Add GitHub connector
3. Display name must be exactly: GitHub
4. Reload this artifact

Tools used: list_pull_requests, list_issues, list_commits

XC Brain MCP not connected

Persistent real-file storage — notes, tasks, captures saved to repo files instead of localStorage.

Setup (run locally):

cd brain/mcp-server node index.js

Then add as connector in claude.ai with name: XC Brain
